🔍 セミナー追加エラーのデバッグ

📋 問題の説明

管理画面のセミナー管理から「追加」ボタンをクリックすると、「セミナーデータの取得に失敗しました」というエラーが表示される問題について調査します。

期待される動作：
• /admin/seminars/new にアクセス
• 新規作成フォームが表示される
• エラーメッセージは表示されない

🚀 Step 1: ルーティングテスト

正しいルートが設定されているか確認します。

🔧 Step 2: Mock Server のデバッグ

mockServer.getSeminar() が 'new' IDで呼ばれたときの動作を確認します。

🎯 Step 3: 直接アクセステスト

以下のリンクから直接アクセスして動作を確認します。

💡 Step 4: コンソールでデバッグ

新規作成画面を開いてから、ブラウザのコンソールで以下のコマンドを実行してください：

// Vue コンポーネントを確認 const app = document.querySelector('#app').__vue__; let seminarEditForm = null; // SeminarEditForm を探す function findSeminarEditForm(vm) { if (vm.$options.name === 'SeminarEditForm') { seminarEditForm = vm; return; } if (vm.$children && vm.$children.length) { vm.$children.forEach(child => { if (!seminarEditForm) findSeminarEditForm(child); }); } } findSeminarEditForm(app); if (seminarEditForm) { console.log('✅ SeminarEditForm found!'); console.log('Route params:', seminarEditForm.$route.params); console.log('isNew:', seminarEditForm.isNew); console.log('loading:', seminarEditForm.loading); console.log('error:', seminarEditForm.error); console.log('formData:', seminarEditForm.formData); } else { console.log('❌ SeminarEditForm not found'); }
