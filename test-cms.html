CMS管理画面 更新機能テスト

📝 テスト手順

以下の手順に従って、CMS管理画面の更新機能をテストしてください。

ステップ 1: アプリケーションの起動

1. ターミナルで npm start を実行してVueアプリケーションを起動

2. ブラウザで http://localhost:8081 を開く

3. 管理画面にアクセス: /admin/login

ステップ 2: セミナー管理のテスト

1. セミナー管理画面を開く

2. 「追加」ボタンをクリックして新規セミナーを作成

3. フォームに情報を入力して保存

4. 一覧画面に戻って、新しいセミナーが表示されることを確認

ステップ 3: お知らせ管理のテスト

1. お知らせ管理画面を開く

2. 新規お知らせを作成

3. 既存のお知らせを編集

4. ニュースページで反映を確認

ステップ 4: 刊行物管理のテスト

1. 刊行物管理画面を開く

2. 新規刊行物を追加

3. 既存の刊行物情報を更新

ステップ 5: HomePage.vueでの表示確認

1. CMS管理画面で新しいデータを追加

2. ホームページを開く（またはリロード）

3. 最新のニュース、セミナー数、刊行物数が自動的に表示されることを確認

ステップ 6: データの永続性確認

ページをリロードして、データが保持されているか確認します。
