🔍 Vue Component デバッグツール

1. ブラウザコンソールで以下を実行してください

HomePage (http://localhost:8081) を開いて、F12でコンソールを開き、以下のコマンドを1つずつ貼り付けて実行：

// Step 1: Vueコンポーネントを取得 const app = document.querySelector('#app').__vue__; console.log('Vue app:', app); // Step 2: HomePageコンポーネントを探す const findComponent = (root, name) => { if (root.$options.name === name) return root; for (let child of root.$children) { const found = findComponent(child, name); if (found) return found; } return null; }; const homePage = findComponent(app, 'HomePage'); console.log('HomePage component:', homePage); // Step 3: データの内容を確認 if (homePage) { console.log('=== Current Data Values ==='); console.log('seminar:', homePage.seminar); console.log('publications1:', homePage.publications1); console.log('infomation1:', homePage.infomation1); console.log('date1:', homePage.date1); console.log('hotInfomationVol319:', homePage.hotInfomationVol319); console.log('date2:', homePage.date2); console.log('text123:', homePage.text123); console.log('==========================='); } else { console.error('HomePage component not found!'); }

2. 手動でデータを更新

もしデータが古い場合、以下のコマンドで直接更新してみてください：

// HomePageコンポーネントのデータを直接更新 if (homePage) { // 直接データを変更 homePage.seminar = '【テスト】セミナー(99件)'; homePage.publications1 = '【テスト】刊行物(88件)'; homePage.infomation1 = '【テスト】お知らせ(77件)'; homePage.date1 = '2025.12.31'; homePage.hotInfomationVol319 = '【テスト】これは手動更新のテストです'; // 強制的に再描画 homePage.$forceUpdate(); console.log('✅ データを手動で更新しました！'); console.log('画面に【テスト】が表示されているか確認してください'); } else { console.error('❌ HomePageコンポーネントが見つかりません'); }

3. 表示要素を直接確認

DOM要素の内容を確認：

// DOM要素を直接確認 console.log('=== DOM Elements ==='); const seminarEl = document.querySelector('.seminar'); const publicationsEl = document.querySelector('.publications'); const infomationEl = document.querySelector('.infomation'); console.log('Seminar element text:', seminarEl ? seminarEl.textContent : 'Not found'); console.log('Publications element text:', publicationsEl ? publicationsEl.textContent : 'Not found'); console.log('Infomation element text:', infomationEl ? infomationEl.textContent : 'Not found'); // ニュース項目を確認 const dates = document.querySelectorAll('.date'); console.log('Date elements count:', dates.length); dates.forEach((el, i) => { console.log(`Date ${i+1}:`, el.textContent); });

4. 結果を教えてください

  • Step 1-3 の結果： データの値は更新されていますか？
  • 手動更新の結果： 【テスト】の文字が画面に表示されましたか？
  • DOM要素の結果： 要素は見つかりましたか？

⚠️ 重要：
もし手動更新で【テスト】が表示された場合 → loadLatestData()メソッドの問題
もし手動更新でも表示されない場合 → Vue.jsのリアクティビティまたはテンプレートの問題
